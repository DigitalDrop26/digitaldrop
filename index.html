Caricamento…

Se resta solo questa schermata o una pagina bianca, molto probabilmente stai aprendo il file index.html direttamente dal disco. Con React e Vite il browser non esegue il codice in quel modo.

Fai così: nella cartella del progetto esegui nel terminale:

npm install
npm run dev

Poi apri l’indirizzo che compare (di solito http://localhost:5173). Per vedere la versione di produzione: npm run build e npm run preview.

Online l’indirizzo del sito è https://digitaldrop26.github.io/digitaldrop/ (serve il suffisso /digitaldrop/).
